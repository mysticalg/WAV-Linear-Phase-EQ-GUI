Why it exists

For exported WAVs that feel too rigid straight out of the render.

The free build covers the core mastering pass: linear-phase high-pass and low-pass filtering, export controls, preview playback, waveform views, and final limiting.

The premium unlock opens Pitch, Tape, Compressor, Texture, Stereo, Humanize, and Offset tools. Unlocking happens inside the app so the code can be bound to your install ID.

Workflow

  1. Add one WAV or a batch folder.
  2. Dial in the linear-phase filter and export settings.
  3. Preview the result with waveform, spectrum, and spectrogram feedback.
  4. Use premium FX when you want extra movement, glue, or texture.
